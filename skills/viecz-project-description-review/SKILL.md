# Viecz Project Description Review

Review the Viecz REDACTED_UNIVERSITY I&E 2025 project description for consistency and quality.

## Tone & Voice

- **Calm explainer** — like explaining to a friend over coffee. Not selling, just describing.
- No AI-sounding bullet lists. Write in flowing paragraphs when possible.
- Vietnamese text must sound natural — not translated from English.
- Personal stories use raw, specific details (temperature, weight, time of day). No generic "many students face difficulties."
- Never overcomplicate or dream about features that don't exist yet. Describe what the product **actually does today**.

## Content Rules

- Every claim must be backed by a reference in `docs/general/THÔNG TIN CUỘC THI/references/` or data from YouTrack PROJECT-A-1.
- No personal info (emails, tokens, real names of team members beyond what the competition requires).
- "Trao đổi" (exchange), not altruism — everyone gets something (money, skill, help back, gratitude).
- Viecz is a **coordination** platform, not a communication or social network app.
- Android app status: "thử nghiệm nội bộ" (internal testing). iOS: future plan. Never overclaim.
- Escrow payment is optional — tasks can be free/volunteer.

## Structure Checks

- Sections must follow the REDACTED_UNIVERSITY I&E template strictly (7 sections).
- Each section flows logically into the next — no repetition across sections.
- Technical details belong in Section 4.3, not scattered elsewhere.
- Vision/impact content belongs in Section 7, not in feature descriptions.
- Media placeholders (`<!-- [MEDIA: ...] -->`) must describe what goes there specifically.

## Formatting

- A4, font 14, Times New Roman (for final PDF — markdown draft notes this).
- Tables: keep them compact. Max 5-6 rows per table. Cut to essentials.
- No emoji in the document.
- Currency in VND with dot separators (50.000 VND, not 50,000 or 50000).

## Review Checklist

When reviewing, check each section for:
1. Does it sound human? Read it aloud — if it sounds like a chatbot wrote it, rewrite.
2. Is every fact verifiable from the references?
3. Does it stay grounded in what exists, not what we wish existed?
4. Is the Vietnamese natural, not stilted or overly formal?
5. Are there any contradictions with other sections?
6. Is the section the right length — not too bloated, not too thin?
